# 澳洲私人医院保险购买决策成本公式

## 概述

本文档定义了一个用于计算"立即购买 vs 延迟购买"澳洲私人医院保险的成本差异公式。该公式帮助用户在纯经济角度上评估是否应该立即购买保险，或者可以延迟购买。

---

## 核心公式

```
净额外成本 = P × X × 0.2 + 收入 × MLS税率 × X - P × (1 + L₀) × X
```

### 公式说明

该公式计算的是：在同一时间跨度内，采用"**等待X年后再购买保险**"的策略，相比"**现在立即购买保险**"的策略，在纯经济成本上的差异。

- **正数**：等待X年会多花钱 → 建议立即购买
- **负数**：等待X年反而省钱 → 从纯经济角度，等待划算
- **零**：两种策略成本相同 → 临界点

---

## 参数定义

### P - 基础年保费
- **定义**：假设没有任何Loading情况下的年度保险费用
- **典型值**：$1,500 - $3,000（根据保险等级）
  - Bronze（基础）：$1,500 - $2,000
  - Silver（中级）：$2,000 - $2,500
  - Gold（高级）：$2,500 - $3,500
- **注意**：不同年龄、保险公司、覆盖范围会影响P值

### X - 延迟购买年数
- **定义**：从当前时刻延迟多少年后再购买保险
- **取值范围**：1 - 30年
- **常用值**：1, 3, 5, 8, 10年

### L₀ - 当前Loading百分比
- **定义**：如果现在购买保险，需要承担的Lifetime Health Cover (LHC) Loading
- **计算公式**：`L₀ = (当前年龄 - 基准年龄) × 2%`
- **基准年龄**：
  - 澳洲出生/儿童移民：30岁（31岁生日的7月1日）
  - 成年后移民：获得Medicare后12个月的宽限期结束日
- **取值范围**：0% - 70%（最高上限）
- **示例**：
  - 25岁：L₀ = 0%
  - 35岁：L₀ = 10%（晚了5年）
  - 42岁（39岁拿Medicare）：L₀ = 4%（晚了2年）
  - 50岁：L₀ = 40%（晚了20年）

### 收入 - 用于计算MLS的年收入
- **定义**：用于Medicare Levy Surcharge (MLS)计算的"Income for MLS purposes"
- **包括**：
  - Taxable income（应税收入）
  - Reportable fringe benefits（报告的附加福利）
  - Reportable superannuation contributions（报告的养老金供款，如salary sacrifice）
  - Total net investment loss（净投资损失）
- **不包括**：
  - 雇主强制性Super Guarantee Contribution (SGC)

### MLS税率 - Medicare Levy Surcharge税率
- **定义**：因没有私人医院保险而需额外缴纳的税率
- **2024-25财年税率表**：

#### 单身
| 收入范围 | MLS税率 |
|---------|--------|
| ≤ $97,000 | 0% |
| $97,001 - $113,000 | 1.0% |
| $113,001 - $151,000 | 1.25% |
| > $151,000 | 1.5% |

#### 家庭
| 收入范围 | MLS税率 |
|---------|--------|
| ≤ $194,000 | 0% |
| $194,001 - $226,000 | 1.0% |
| $226,001 - $302,000 | 1.25% |
| > $302,000 | 1.5% |

- **特殊说明**：
  - 家庭每增加一个孩子（除前两个），门槛增加$1,500
  - 大部分Temporary Resident (TR)身份不需要交MLS
  - 检查payslip上是否扣除Medicare Levy来确认

## 公式组成部分详解

### 1. P × X × 0.2 - 未来Loading增加成本

**含义**：因延迟X年购买，导致未来10年Loading期间多支付的总成本。

**推导**：
- 延迟X年，Loading增加：`X × 2%`
- Loading持续时间：10年（连续持有10年后消失）
- 多付成本：`P × (X × 2%) × 10年 = P × X × 0.2`

**示例**：
- 基础保费P = $2,000
- 延迟3年购买
- Loading增加：3 × 2% = 6%
- 10年多付：$2,000 × 6% × 10 = $1,200

### 2. 收入 × MLS税率 × X - 延迟期间的MLS支出

**含义**：延迟X年期间，因没有保险而每年需要额外缴纳的Medicare Levy Surcharge总和。

**计算**：
- 每年MLS = 收入 × MLS税率
- X年累计 = 收入 × MLS税率 × X

**示例**：
- 收入$120,000（单身），MLS税率1.25%
- 延迟3年
- 总MLS = $120,000 × 1.25% × 3 = $4,500

### 3. -P × (1 + L₀) × X - 延迟期间省下的保费

**含义**：延迟X年不买保险，省下的保费支出（负数表示节省）。

**计算**：
- 如果现在买，每年需付：P × (1 + L₀)
- X年累计：P × (1 + L₀) × X
- 因为是省下的，所以加负号

**示例**：
- 基础保费P = $2,000
- 当前Loading L₀ = 4%
- 延迟3年
- 省下保费 = $2,000 × 1.04 × 3 = $6,240

## 使用场景

### 场景1：年轻人（25-30岁）评估是否要在30岁前购买

**参数设置**：
- P = $2,000
- L₀ = 0%（还没到30岁）
- X = 1, 2, 3, 5年
- 收入和MLS税率根据实际情况

**决策要点**：
- 如果收入低于MLS门槛，延迟通常更划算
- 30岁前最后一刻购买可避免Loading

### 场景2：移民/PR评估购买时机

**参数设置**：
- P = $2,000
- L₀ = 根据获得Medicare时间计算
- X = 延迟年数
- 收入和MLS税率

**决策要点**：
- 12个月宽限期内购买不产生Loading
- TR身份通常不需要交MLS
- 如果计划长期留澳，建议早买锁定Loading

### 场景3：中年人（40-50岁）评估是否值得购买

**参数设置**：
- P = $2,000 - $2,500（年龄越大保费可能略高）
- L₀ = 已有的Loading（通常10%-40%）
- X = 1, 3, 5年
- 收入和MLS税率

**决策要点**：
- 已有Loading的情况下，每延迟1年Loading继续增加
- 健康风险开始上升，风险成本需重点考虑
- 即使经济上"不划算"，保障需求可能更重要

## 计算示例

### 示例1：年轻高收入者

**情况**：
- 28岁，单身
- 年收入：$120,000
- MLS税率：1.25%
- 基础保费：$2,000
- 当前Loading：0%

**问题**：现在买 vs 2年后（30岁）买？

```
净额外成本 = 2000 × 2 × 0.2 + 120000 × 1.25% × 2 - 2000 × 1 × 2
          = 800 + 3000 - 4000
          = -$200
```

**结论**：延迟2年省$200（但要注意30岁deadline）

---

### 示例2：移民TR身份

**情况**：
- 42岁，39岁获得Medicare
- 家庭收入：$150,000
- MLS税率：0%（低于家庭门槛$194,000）
- 基础保费：$2,000
- 当前Loading：4%（42-40=2年）

**问题**：现在买 vs 3年后（45岁）买？

```
净额外成本 = 2000 × 3 × 0.2 + 0 - 2000 × 1.04 × 3
          = 1200 + 0 - 6240
          = -$5,040
```

**结论**：延迟3年省$5,040（纯经济角度）

**但需考虑**：
- 42-45岁健康风险上升
- 如果需要手术，自费或等待成本远超$5,040
- 计划长期留澳，建议购买锁定Loading

---

### 示例3：高收入中年人

**情况**：
- 45岁
- 单身年收入：$180,000
- MLS税率：1.5%
- 基础保费：$2,000
- 当前Loading：30%（晚了15年）

**问题**：现在买 vs 5年后（50岁）买？

```
净额外成本 = 2000 × 5 × 0.2 + 180000 × 1.5% × 5 - 2000 × 1.30 × 5
          = 2000 + 13500 - 13000
          = +$2,500
```

**结论**：延迟5年多花$2,500，应该立即购买

## 临界点计算

### 问题：什么收入水平下，延迟购买开始"不划算"？

当净额外成本 = 0时：

```
P × X × 0.2 + 收入 × MLS税率 × X - P × (1 + L₀) × X = 0
收入 × MLS税率 × X = P × (1 + L₀) × X - P × X × 0.2
收入 × MLS税率 = P × (1 + L₀ - 0.2)
收入 = P × (1 + L₀ - 0.2) / MLS税率
```

### 临界收入表

假设P = $2,000，MLS税率 = 1%：

| 当前Loading (L₀) | 临界收入 |
|-----------------|---------|
| 0% | $168,000 |
| 4% | $176,800 |
| 10% | $188,000 |
| 20% | $208,000 |
| 30% | $228,000 |

**解读**：
- 收入超过临界值 → 立即购买划算
- 收入低于临界值 → 延迟购买纯经济上更优

## 适用条件

### ✅ 公式适用于

1. **任何年龄段**（18-70岁）
2. **任何收入水平**
3. **任何延迟年数**（1-30年）
4. **所有Loading情况**（0%-70%）
5. **快速经济对比**

### ⚠️ 重要前提假设

公式基于以下假设：

1. **基础保费P不变**
   - 实际会因年龄增长、通胀上涨3-5%/年
   - 可能影响长期（5年+）预测准确性

2. **收入和MLS税率不变**
   - 实际收入可能变化
   - 可能跨越MLS门槛

3. **Loading上限70%**
   - 如果L₀ + X×2% > 70%，需要调整计算

4. **持有10年以上**
   - 假设购买后会持续持有10年直到Loading消失
   - 如果中途取消，Loading不会消失

5. **健康状况不变**
   - 不考虑pre-existing conditions影响

---

## 公式的局限性

### ❌ 未考虑的重要因素

#### 1. 医疗风险成本（最重要！）

公式**不包括**延迟X年期间没有保险的潜在医疗支出：

| 手术类型 | 自费成本 | 公立等待时间 |
|---------|---------|------------|
| 膝关节置换 | $20,000-$30,000 | 6-18个月 |
| 髋关节置换 | $25,000-$35,000 | 6-18个月 |
| 心脏支架 | $15,000-$25,000 | 急诊快，择期6-12个月 |
| 疝气修复 | $8,000-$12,000 | 3-9个月 |
| 胆囊切除 | $10,000-$15,000 | 6-12个月 |
| 白内障手术 | $3,000-$5,000/眼 | 6-24个月 |

**任何一次手术的成本可能远超纯经济计算的差额！**

#### 2. 等待期延迟

- 购买保险后有等待期（2-12个月）
- 延迟X年 = 实际保障延迟X+1年
- 机会成本难以量化

#### 3. 保费增长

- 随年龄增长，基础保费P会提高
- 40岁的P vs 50岁的P可能差20-30%
- 通货膨胀导致保费每年上涨

#### 4. 健康状况变化

- 延迟期间可能出现健康问题
- 可能导致pre-existing conditions除外
- 影响保险承保或增加条款

#### 5. 家庭情况变化

- 收入增长可能跨越MLS门槛
- 结婚、生子改变家庭结构
- 需要重新评估

#### 6. 心理和生活质量

- 有保险带来的安心感
- 生病时不用担心经济负担
- 可以选择医生和医院
- 这些价值无法量化

## 决策建议框架

### 综合决策矩阵

| 净额外成本 | 年龄 | 健康状况 | MLS情况 | 建议 |
|-----------|------|---------|---------|------|
| **>$3,000** | 任何 | 任何 | 任何 | **立即买** - 延迟很亏 |
| **$0~$3,000** | <35 | 健康 | 不交MLS | 可以等 - 但注意30岁deadline |
| **$0~$3,000** | 35-45 | 健康 | 不交MLS | 谨慎评估 - 考虑风险 |
| **$0~$3,000** | >45 | 健康 | 不交MLS | **建议买** - 风险上升 |
| **$0~$3,000** | 任何 | 有问题 | 任何 | **立即买** - 风险高 |
| **$0~$3,000** | 任何 | 任何 | 交MLS | **立即买** - 已接近临界点 |
| **负数（省钱）** | <30 | 健康 | 不交MLS | 可以等 - 但30岁前买 |
| **负数（省钱）** | 30-40 | 健康 | 不交MLS | 谨慎决定 - 考虑长期 |
| **负数（省钱）** | >40 | 健康 | 不交MLS | **建议买** - 风险>收益 |
| **负数（省钱）** | 任何 | 有问题 | 任何 | **立即买** - 风险成本高 |

### 决策流程图

```
开始
  ↓
计算净额外成本
  ↓
是否 > $3,000？
  ├─ 是 → 立即购买（延迟很亏）
  └─ 否 ↓
是否需要交MLS？
  ├─ 是 → 立即购买（已在临界点）
  └─ 否 ↓
年龄是否 > 40？
  ├─ 是 → 建议购买（风险上升）
  └─ 否 ↓
健康状况如何？
  ├─ 有问题 → 立即购买（风险高）
  └─ 健康 ↓
是否长期留澳？
  ├─ 是 → 建议购买（锁定Loading）
  └─ 否 → 可以等待（但注意时间点）
```

### 技术实现建议

#### 核心函数

```javascript
/**
 * 计算延迟购买的净额外成本
 * @param {number} P - 基础年保费
 * @param {number} X - 延迟年数
 * @param {number} L0 - 当前Loading比例（小数形式，如0.04）
 * @param {number} income - 年收入
 * @param {number} mlsRate - MLS税率（小数形式，如0.0125）
 * @returns {number} 净额外成本（正数表示延迟更贵）
 */
function calculateDelayCost(P, X, L0, income, mlsRate) {
  const loadingCost = P * X * 0.2;
  const mlsCost = income * mlsRate * X;
  const savedPremium = P * (1 + L0) * X;
  
  return loadingCost + mlsCost - savedPremium;
}
```

#### 辅助函数

```javascript
/**
 * 根据年龄和获得Medicare时间计算当前Loading
 * @param {number} currentAge - 当前年龄
 * @param {number} medicareAge - 获得Medicare时的年龄（如果是移民）
 * @param {boolean} isImmigrant - 是否是移民
 * @returns {number} Loading比例（小数形式）
 */
function calculateCurrentLoading(currentAge, medicareAge, isImmigrant) {
  let baseAge = isImmigrant ? medicareAge + 1 : 30;
  let yearsLate = Math.max(0, currentAge - baseAge);
  let loading = Math.min(yearsLate * 0.02, 0.70); // 最高70%
  
  return loading;
}

/**
 * 根据收入和家庭状况确定MLS税率
 * @param {number} income - 年收入
 * @param {boolean} isFamily - 是否家庭
 * @param {number} numChildren - 孩子数量（超过2个的部分）
 * @returns {number} MLS税率（小数形式）
 */
function determineMlsRate(income, isFamily, numChildren = 0) {
  // 2024-25财年的门槛
  const thresholds = isFamily 
    ? {
        tier0: 194000 + (numChildren * 1500),
        tier1: 226000 + (numChildren * 1500),
        tier2: 302000 + (numChildren * 1500)
      }
    : {
        tier0: 97000,
        tier1: 113000,
        tier2: 151000
      };
  
  if (income <= thresholds.tier0) return 0;
  if (income <= thresholds.tier1) return 0.01;
  if (income <= thresholds.tier2) return 0.0125;
  return 0.015;
}
```

#### 数据验证

```javascript
/**
 * 验证输入参数
 */
function validateInputs(age, income, premium, delayYears) {
  const errors = [];

  }

}
```

## 重要免责声明

### ⚠️ 本公式的使用限制

1. **仅供参考**
   - 本公式仅提供纯经济成本对比
   - 不构成财务或医疗建议
   - 实际决策应咨询专业人士

2. **简化假设**
   - 公式基于多项简化假设
   - 实际情况可能更复杂
   - 个人情况千差万别

3. **风险成本**
   - 公式不包括医疗风险成本
   - 健康问题的经济影响可能巨大
   - 不应仅基于公式结果做决定

4. **数据时效性**
   - MLS门槛和税率每年可能调整
   - 保险政策可能变化
   - 使用前请确认最新数据

5. **个人决策**
   - 每个人情况不同
   - 需要综合考虑多个因素
   - 经济只是决策因素之一

## 附录：快速参考

### 公式速查

```
净额外成本 = P × X × 0.2 + 收入 × MLS税率 × X - P × (1 + L₀) × X

其中：
- P：基础年保费（$1,500-$3,000）
- X：延迟年数
- L₀：当前Loading（= (当前年龄 - 基准年龄) × 2%）
- MLS税率：0%/1%/1.25%/1.5%
```

### MLS门槛速查（2024-25）

| 类型 | Tier 0 (0%) | Tier 1 (1%) | Tier 2 (1.25%) | Tier 3 (1.5%) |
|-----|-------------|-------------|----------------|---------------|
| 单身 | ≤$97k | $97k-$113k | $113k-$151k | >$151k |
| 家庭 | ≤$194k | $194k-$226k | $226k-$302k | >$302k |

### 决策快速指南

- **有健康问题** → 立即买
